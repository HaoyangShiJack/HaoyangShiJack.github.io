#### 3.Poisson Distribution
the Poisson distribution has only one single parameter $\lambda$ and a probability mass function given by: $p(k) = \frac{e^{-\lambda}\lambda^k}{k!}$ for $k\geq 0.$ The parameter $\lambda$ represents a mean such as the number of hits of a website in one minute, or the number of mispelled words in a document. Thus $p(k)$ represents the probability the number of events occuring is $k$ given that the average number events that occur is $\lambda$.(==$P(k)$å¯ä»¥é€šä¿—çš„è®¤ä¸ºå°±æ˜¯åœ¨ä¸€æ—¶é—´é‡Œæ¥å…¬äº¤è½¦çš„æ¦‚ç‡ï¼Œä¾‹å¦‚å‘Šè¯‰æ¯ä¸€å°æ—¶æœ‰ä¸¤è¾†å¤§å·´ï¼Œé‚£ä¹ˆå¦‚æœæŠŠ1ä¸ªå°æ—¶å½“ä½œå•ä½ï¼Œé‚£ä¹ˆ$\lambda=2$==ï¼Œæ‰€ä»¥å…¬å¼ä¹Ÿå¯ä»¥çœ‹ä½œæ˜¯$P_t(k)= \frac{e^{-\lambda t}(\lambda t) ^k}{k!}$) å½“ $n$ is large, $p$ is small, and $np \approx \lambda$.
